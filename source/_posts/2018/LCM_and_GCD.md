## 實作

```csharp
//// GCD
static Func<int, int, int> GCD = (numA, numB) => {
        return numB == 0 ? numA : GCD(numB, numA % numB);
    };
//// LCM
static Func<int, int,int> LCM = (numA,numB) => {
            return numA * numB / GCD(numA, numB);
        } ;    
```
